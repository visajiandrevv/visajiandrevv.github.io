整型数据范围

byte：-128 ~ 127 (1个字节，$-2^{7}至(2^{7}-1)$)
short: -32768 ~ 32767 (2个字节，$-2^{15}至(2^{15}-1)$)
int: -2147483648 ~ 2147483647 (4个字节，$-2^{31}至(2^{31}-1)$)
long: -9223372036854775808 ~ 9223372036854775807 (8个字节，$-2^{63}至(2^{63}-1)$)
